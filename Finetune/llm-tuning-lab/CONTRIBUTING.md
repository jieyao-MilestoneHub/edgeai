# 貢獻指南

感謝你對 LLM Tuning Lab 的興趣！

## 如何貢獻

1. Fork 本專案
2. 創建新分支 (`git checkout -b feature/amazing-feature`)
3. 提交更改 (`git commit -m 'Add amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 開啟 Pull Request

## 程式碼風格

- 使用 Black 格式化
- 遵循 PEP 8
- 添加型別註解
- 撰寫 docstring

## 測試

```bash
pytest tests/ -v
```

## 問題回報

請使用 GitHub Issues 回報問題。
